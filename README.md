# PNG-Streamer

PNG Streamer - инструмент для стримеров, которые хотят использовать на своих стримах различные png-изображения в качестве своего аватара.<br>
Среди преимуществ данного инструмента:<br>
- ⚡ Простота - Вы настраиваете профиль посредством выбора отдельных изображений для конкретных ситуаций, выбираете микрофон (галочка Шумоподавление доступна тоже) и на этом вя настройка заканчивается.<br>
- 🛋️ Удобство - Вам больше не надо держать окно приложения постоянно открытым для того, чтобы вырезать содержимое окна в obs. Данное приложение использует веб-сервер на основе FastApi и Uvicorn для отображения модели. Просто скопируйте ссылку адреса виджета и вставьте её источником, как, к примеру, оповещение о Донате. Для предпросмотра можете просто вставить эту ссылку в свой браузер.<br>
- ⚙️ Продвинутость - Обычно, в подобных приложениях вы можете выбрать изображения только для случаем разговора и/или моргания. Я считаю этого недостаточно, поэтому в данном приложении, помимо моргания, вы можете настроить изображение в зависимости от громкости вашей речи! При этом, вы сами указываете пороги срабатывания, а не устанавливаете изображения в заранее заготовленные места.
